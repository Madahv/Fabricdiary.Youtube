## Step 1: GitHub Account banao (agar nahi hai)
1. github.com pe jao
2. Sign up karo (free)

---

## Step 2: Code GitHub pe daalo

### Option A: GitHub Desktop (Easy, Recommended for beginners)
1. desktop.github.com se GitHub Desktop download karo
2. "New Repository" → Name: yt-seo-tool
3. Is folder ki saari files wahan copy karo
4. "Commit to main" → "Publish Repository"

### Option B: Direct GitHub Upload
1. github.com → New Repository → Name: yt-seo-tool → Create
2. "Upload files" click karo
3. Ye saari files drag & drop karo:
   - app.py
   - requirements.txt
   - render.yaml
   - Procfile
   - templates/ folder (index.html ke saath)
4. "Commit changes" click karo

---

## Step 3: Render pe Deploy karo (FREE)

1. render.com pe jao → Sign Up (GitHub se login karo)
2. "New +" → "Web Service"
3. Apna GitHub repo select karo (yt-seo-tool)
4. Settings:
   - Name: yt-seo-extractor (ya kuch bhi)
   - Environment: Python 3
   - Build Command: `pip install -r requirements.txt`
   - Start Command: `gunicorn app:app`
   - Plan: **Free**
5. "Create Web Service" click karo
6. 2-3 minute wait karo — deploy ho jayega!
